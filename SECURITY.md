# Security Policy

## Scope

This policy covers all repositories under the [Loopress GitHub organization](https://github.com/loopress), including but not limited to:

- The `@loopress/cli` CLI
- The companion WordPress plugin `loopress`

Vulnerabilities affecting third-party dependencies should preferably be reported directly to their respective maintainers, but you're welcome to report them to us as well if you're unsure of the right channel.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| Latest stable release | ✅ |
| Previous releases | ❌ |

## Reporting a Vulnerability

Please report any vulnerability **only via GitHub Security Advisories**: the *Security > Report a vulnerability* tab of the repository.

**Please do not open a regular public issue** to report a security flaw: this would expose the vulnerability before a fix is available. Security Advisories allows for a private report, visible only to maintainers until coordinated disclosure.

In your report, please include:

- A description of the vulnerability and its potential impact
- Steps to reproduce it (proof of concept if possible)
- The affected version (CLI and/or plugin)
- Your name or handle if you'd like to be credited

### What to Expect

- **Acknowledgement**: within 72 business hours
- **Initial assessment**: within 7 days (confirmation, severity, scope)
- **Fix**: timeline communicated based on severity, generally within 30 to 90 days
- We'll keep you updated on progress at each step

## Coordinated Disclosure

We ask for a reasonable amount of time before any public disclosure, to allow a fix to be developed and deployed. Once the fix is released, we're happy to credit whoever reported the issue (unless they'd prefer otherwise).

## Safe Harbor

Any research conducted in good faith, in accordance with this policy, without unauthorized access to others' data or service disruption, will not be subject to legal action on our part.

## Contact

GitHub Security Advisories for this repository (*Security > Report a vulnerability* tab).
